# ⚡ 빠른 시작 가이드

## 1단계: 서버 시작

터미널에서 프로젝트 폴더로 이동한 후:

### 🐍 Python 사용 (가장 간단)
```bash
python3 server.py
```

### 📦 Node.js 사용
```bash
npm install  # 최초 1회만
npm start
```

---

## 2단계: 브라우저 접속

서버가 시작되면 자동으로 브라우저가 열립니다.
또는 수동으로 접속: **http://localhost:8000/home.html**

---

## 3단계: 확인

1. home.html에서 지역 마커 클릭 → 소원 입력
2. "업로드" 버튼 클릭
3. 라디얼 메뉴에서 "소원마당" 클릭
4. soma2.html에서 같은 지역 버튼 클릭
5. ✅ 저장한 소원이 표시되는지 확인!

---

## ⚠️ 중요

- **절대 `file://`로 파일을 직접 열지 마세요!**
- **반드시 `http://localhost:8000`으로 접속해야 합니다**
- 서버가 실행 중일 때만 접속 가능합니다
